# My First Cypress Automation

Project Description

This project is an end-to-end test automation built with Cypress, focused on validating critical user flows of a web application.

**The main features covered in this project include:**

> Login Feature: Automated tests to ensure the authentication process works correctly with valid and invalid credentials.
> Personal Information Update: Tests that validate the update of user personal data, ensuring the information is saved and displayed properly after changes.

The project follows good automation practices, aiming for readable, maintainable, and scalable test code. It is a great example of how Cypress can be used to test real user scenarios and improve application reliability.

## Installation:
```bash
npm install
```
> **NOTE**
>
> it is necessary to have **node** installed
>
## Running:

You open the Cypress UI
 ```bash
npx cypress open
```
Run the automation by command line
```bash
npx cypress run
```
